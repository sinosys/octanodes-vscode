# Changelog

## [0.2.0] - 2026-07-02

- **상태바 + 자동 새로고침** — 하단 상태바에 미완료 일감 수 표시, 주기적 폴링(`octanodes.refreshInterval`, 기본 120초)으로 새 일감·상태 변경 시 알림 토스트
- **일감 검색** — 툴바 🔍 또는 명령 팔레트에서 제목·#번호로 전사 일감 라이브 검색 → 상세 열기
- **상세 편집 강화** — 상세 웹뷰 툴바에서 상태·우선순위·담당자·마감일 변경, 제목·본문 인라인 수정, 브라우저에서 열기
- **트리 그룹 기준** — 프로젝트별/상태별/우선순위별 그룹 전환, 그룹 내 우선순위 정렬
- **필터·그룹 영속화** — 상태 필터와 그룹 기준을 재시작 후에도 유지

## [0.1.1] - 2026-07-01

- 트리뷰 일감 라벨에 `#번호`를 제목 앞에 표시
- 상태 필터에 **미완료만 (종료·반려 제외)** 추가 및 기본값으로 설정
- 상태 필터 선택 시 현재 항목 표시(✓) + 트리뷰 상단 필터 안내
- 공개 저장소용 `homepage`·`bugs` 링크 추가

## [0.1.0] - 2026-07-01

첫 공개 버전.

- 내 일감 트리뷰 (프로젝트별 그룹, 상태 필터)
- 일감 상세 웹뷰 (본문·단계·댓글, 댓글 추가)
- 일감 등록 / 상태 변경 / 댓글 추가 명령
- PAT 로그인 (SecretStorage 저장), 로그아웃
- `octanodes.baseUrl` 설정 (온프렘/로컬 지원)
